# A Giter8 template for sbt with scalafmt.

Create a new project with supporting scalafmt(formatting on compile)
 using giter8:

## preparing

```sh
$ brew install giter8
```

## usage

- for giter8

```sh
$ g8 j5ik2o/sbt-with-scalafmt.g8
```

- for sbt new

```sh
$ sbt new j5ik2o/sbt-with-scalafmt.g8
```
